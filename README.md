# Claude Code Skills

我的 Claude Code 自定义技能集合。

## 技能列表

| 技能 | 说明 |
|------|------|
| **audio-transcription** | 音频/视频转文字，支持 mp3, wav, m4a, mp4 等格式，基于 Whisper API |
| **video-downloader** | 在线视频下载，支持 YouTube、Bilibili、Twitter/X、抖音等平台 |
| **skill-manager** | 技能管理器，扫描、分类、健康检查、同步飞书多维表格 |

## 使用方式

将技能目录复制到 `~/.claude/skills/` 下即可使用。
