# 九宫格故事版提示词编写指南

## 核心原则

九宫格故事版是整个短剧的基础，提示词质量直接决定：
- 角色是否一致
- 风格是否统一
- 故事是否连贯

## 提示词结构

### 基础模板

```
创建一个3x3的九宫格电影分镜故事版。

故事：[故事概要，2-3句话]

主角设定：
- 外貌：[详细的外貌描述，越具体越好]
- 服装：[服装描述]
- 年龄：[年龄范围]

画面风格：[风格描述]

九宫格分镜安排（从左到右，从上到下为K1-K9）：
K1: [第一个画面描述]
K2: [第二个画面描述]
K3: [第三个画面描述]
K4: [第四个画面描述]
K5: [第五个画面描述]
K6: [第六个画面描述]
K7: [第七个画面描述]
K8: [第八个画面描述]
K9: [第九个画面描述]

要求：
1. 主角在所有9格中外貌完全一致
2. 整体色调和光线保持统一
3. 每格画面饱满清晰
4. 无文字或对话框
```

### 风格描述参考

**电影感**
```
电影质感画面，35mm胶片色调，柔和的自然光，浅景深，温暖的色彩调色，专业电影构图
```

**动漫风**
```
日式动漫风格，细腻的线条，明亮的色彩，大眼睛角色设计，精致的背景绘制
```

**写实风**
```
超写实风格，高清摄影质感，自然的光影，真实的皮肤纹理，细节丰富
```

**复古风**
```
80年代复古风格，暖黄色调，胶片颗粒感，怀旧氛围，柔和的边缘
```

**赛博朋克**
```
赛博朋克风格，霓虹灯光，蓝紫色调，未来科技感，高对比度
```

## 分镜编排技巧

### 爆款叙事公式（推荐）

> 💡 **核心原则**：开篇就是悬念，禁止平铺直叙；中间层层反转，剧情紧凑；结尾升华留白，引发思考

**第一幕（K1-K2）：抛出困境/悬念**
- K1: 直接抛出问题/困境/反常现象（让观众瞬间被吸引）
- K2: 建立危机感，让观众不敢眨眼

**第二幕（K3-K5）：反转/颠覆**
- K3: 展开解谜，揭示第一层线索
- K4: 颠覆认知，打破常识
- K5: 第二层反转，层层递进

**第三幕（K6-K7）：高潮**
- K6: 关键揭示/真相大白
- K7: 高潮时刻，情感顶点

**第四幕（K8-K9）：升华/留白**
- K8: 哲学升华，提升立意
- K9: 留白结局，引发思考（让观众想看更多）

### 情感弧线

确保分镜情感有强烈起伏（适合短视频平台）：
```
K1: 困惑/震撼 → K3: 好奇/紧张 → K5: 颠覆/惊讶 → K7: 高潮/情感爆发 → K9: 升华/回味
```

### 经典三幕式结构（备选）

如果故事不适合爆款公式，可使用传统结构：

**第一幕（K1-K3）：建立**
- K1: 环境/背景建立
- K2: 主角出场
- K3: 冲突/问题引入

**第二幕（K4-K6）：发展**
- K4: 主角面对挑战
- K5: 转折点
- K6: 高潮前奏

**第三幕（K7-K9）：解决**
- K7: 高潮
- K8: 结果
- K9: 结局/情感落点

## 角色一致性技巧

### 详细描述外貌

不好的描述：
```
一个年轻女性
```

好的描述：
```
25岁亚洲女性，黑色齐肩直发，杏眼，高挺鼻梁，穿白色衬衫和深蓝色西装外套
```

### 固定特征元素

选择2-3个易识别的特征在所有分镜中保持：
- 标志性发型
- 特殊配饰（眼镜、耳环等）
- 独特服装颜色

## 常见问题解决

### 角色不一致

原因：外貌描述不够具体
解决：添加更多细节描述，包括发型、五官、服装

### 风格跳跃

原因：风格描述含糊
解决：使用具体的风格关键词，参考上方风格描述

### 故事断裂

原因：分镜之间缺乏连接
解决：确保相邻分镜有逻辑关联，使用过渡画面

## 视频提示词编写

生成视频时，每个片段的提示词应该描述：

```json
{
  "name": "scene_01",
  "prompt": "镜头缓慢推进，人物从[K1状态]过渡到[K2状态]，[动作描述]",
  "first_frame": "K1.png",
  "last_frame": "K2.png"
}
```

### 动作描述参考

- 缓慢行走 → "人物缓步向前走"
- 转身 → "人物转身面向镜头"
- 情绪变化 → "表情从平静变为惊讶"
- 环境变化 → "光线从明亮变为昏暗"

## 旁白编写指南

### 风格

- 紧凑有力，每句话必须有信息量
- 配合画面节奏，旁白与画面同步
- 使用现在时增加沉浸感
- 禁止"嗯、啊、那个"等填充词

### 每个分镜旁白长度（适配8秒视频）

| 语言 | 字数/词数 | 朗读时长 |
|------|----------|---------|
| 中文 | **35-50 字** | 约 8 秒 |
| 英文 | **15-25 words** | 约 8 秒 |

> ⚠️ **重要**：旁白时长要填满 8 秒视频，不能有"空等"。每个分镜都要内容饱满。

### 紧凑度要求

- 每句话必须推进故事或传递信息
- 避免重复画面已经表达的内容
- 适合短视频平台的快节奏风格
- 让观众感觉"信息量很大"

### 示例（8秒旁白）

```json
{
  "scene_01": "你此刻觉得自己拥有自由意志，是你决定了手指划过屏幕。但如果我告诉你，这个决定在你意识到之前0.5秒，就已经被大脑做出了呢？",
  "scene_02": "1983年，神经科学家本杰明·利贝特做了一个震惊世界的实验。他发现，在人们自认为做出决定之前，大脑已经产生了准备电位。"
}
```

### 旧版示例（3-5秒，已弃用）

```json
{
  "scene_01": "在这片被遗忘的土地上，她踏出了改变命运的第一步。",
  "scene_02": "前方的道路充满未知，但她的眼神坚定如初。"
}
```

> 注意：旧版示例只有 15-25 字，无法填满 8 秒视频。

## 内容紧凑度要求

> 💡 这是适合自媒体平台的关键要素，直接影响完播率

### 核心原则

1. **每格都要有内容**：禁止"过渡性"空镜头，9 个分镜每个都要有明确的信息点或情感变化
2. **旁白与画面同步**：不能有视频在播但没有旁白的"空等"时刻
3. **快节奏剪辑**：适合短视频平台的观看习惯，3秒无变化观众就会划走
4. **信息密度高**：让观众感觉"学到了东西"或"被震撼到了"

### 检查清单

在完成脚本后，逐一检查：

- [ ] K1 是否在前 3 秒抓住观众注意力？
- [ ] 每个分镜是否都有新信息或情感变化？
- [ ] 旁白是否填满 8 秒？有没有"空等"？
- [ ] 整体节奏是否紧凑？有没有拖沓的地方？
- [ ] 结尾是否让人想看更多？

### 常见问题

| 问题 | 表现 | 解决方案 |
|------|------|---------|
| 开头太平淡 | K1 只是环境介绍 | 改为直接抛出悬念/问题 |
| 中间拖沓 | K4-K6 没有新信息 | 增加反转或颠覆点 |
| 旁白太短 | 每段只有 20 字 | 扩充到 35-50 字 |
| 结尾无力 | K9 只是简单结束 | 增加升华或留白 |

## 发布素材编写指南

> 💡 好的标题和简介能显著提升点击率和推荐量

### 视频标题

**长度**：< 70 字符（太长会被截断）

**技巧类型**：

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念式 | "如果...会怎样？" | "如果你的大脑一直在骗你，你会怎样？" |
| 数字式 | "X%的人不知道..." | "99%的人不知道，你的决定早在0.5秒前就被做出了" |
| 反问式 | "你真的...吗？" | "你真的拥有自由意志吗？" |
| 颠覆式 | "X其实是Y" | "你以为的选择，其实是大脑的骗局" |

### 视频简介

**长度**：100-200 字

**结构**：
1. **核心卖点**（1-2句）：为什么观众要看这个视频
2. **内容概要**（1-2句）：简单介绍会讲什么
3. **引导互动**（1句）：引导点赞、评论、关注

**示例**：
```
你以为的自由意志，可能只是一场精心设计的骗局...

1983年，一个震惊世界的实验揭示了惊人真相：在你"决定"之前，大脑早已做出了选择。这个发现动摇了哲学界千年的根基。

看完这个视频，你会重新审视自己的每一个选择。你觉得自己有自由意志吗？评论区告诉我。

#科普 #大脑 #心理学 #自由意志 #神经科学
```

### 话题标签

**数量**：3-5 个

**选择原则**：
1. **平台热门标签**：如 #科普 #涨知识 #冷知识
2. **垂直领域标签**：如 #心理学 #神经科学 #物理学
3. **内容相关标签**：如 #大脑 #自由意志 #实验

**注意**：不同平台的热门标签不同，需要根据目标平台调整。

## 九宫格提取提示词指南（Nano Banana Pro）

> 💡 用于从九宫格中精确提取单张分镜图并放大，避免偏差和不稳定

### 位置坐标对照表

| 格子 | 位置描述 | 坐标描述 | 英文坐标 |
|------|---------|---------|---------|
| K1 | 左上角 | 第1行第1列 | Row 1, Column 1 |
| K2 | 上方中间 | 第1行第2列 | Row 1, Column 2 |
| K3 | 右上角 | 第1行第3列 | Row 1, Column 3 |
| K4 | 左侧中间 | 第2行第1列 | Row 2, Column 1 |
| K5 | 正中央 | 第2行第2列 | Row 2, Column 2 |
| K6 | 右侧中间 | 第2行第3列 | Row 2, Column 3 |
| K7 | 左下角 | 第3行第1列 | Row 3, Column 1 |
| K8 | 下方中间 | 第3行第2列 | Row 3, Column 2 |
| K9 | 右下角 | 第3行第3列 | Row 3, Column 3 |

### 提示词模板（中文版 - 推荐）

```
【任务】精确提取并放大九宫格中的单张图片

【目标位置】第{行号}行第{列号}列（3x3九宫格中，从左上角开始计数）

【操作要求】
1. 仅提取目标位置的图片，完全删除其他8张图片
2. 将提取的图片放大至全画布（2048x2048）
3. 这是"精确复刻"任务，不是"重新创作"任务

【必须保持不变】
- 人物/角色的五官、发型、服装、姿态
- 场景的布局、物品摆放、光源方向
- 画面的色调、饱和度、明暗对比
- 原始构图的景别和比例
- 所有细节元素的位置关系

【禁止操作】
- 禁止添加原图中没有的元素
- 禁止删除或修改原图中的任何元素
- 禁止改变人物表情或姿态
- 禁止调整色调或风格
- 禁止重新构图或裁切

【输出要求】
- 分辨率：2048x2048
- 单张图片，非九宫格
- 高清晰度，细节丰富
```

### 提示词模板（英文版 - 备选）

```
【Task】Extract and upscale a single panel from the 3x3 grid

【Target】Row {row}, Column {column} (counting from top-left)

【Instructions】
1. Extract ONLY the target panel, remove all other 8 panels
2. Scale up to fill the entire canvas (2048x2048)
3. This is "exact replication", NOT "creative recreation"

【Must Preserve】
- Character features: face, hair, clothing, pose
- Scene layout: object positions, lighting direction
- Visual style: color tone, saturation, contrast
- Original composition and framing
- All spatial relationships between elements

【Do NOT】
- Add any new elements
- Remove or modify any existing elements
- Change facial expressions or body poses
- Adjust color grading or art style
- Re-compose or crop differently

【Output】
- Resolution: 2048x2048
- Single image (not a grid)
- High clarity with rich details
```

### 常见问题和解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 人物外貌变化 | 提示词中缺少强约束 | 添加「禁止改变人物表情或姿态」 |
| 色调偏移 | 使用了「细节添加」等指令 | 移除扩图指令，强调「精确复刻」 |
| 提取错误位置 | 位置描述不够精确 | 使用「第X行第Y列」坐标描述 |
| 画面被重新构图 | 缺少负面约束 | 添加「禁止重新构图或裁切」 |
| 添加了新元素 | AI 发挥创作 | 强调「这不是重新创作任务」 |

### 优化要点总结

1. **用坐标定位**：「第2行第3列」比「中间一排最右边」更精确
2. **强调"复刻"**：明确这是复刻任务，不是创作任务
3. **列出保持元素**：详细列出必须保持不变的元素
4. **添加负面约束**：明确说"不要做什么"
5. **移除扩图指令**：「细节添加和扩图」容易导致偏差
